## About this application - 
This is a comic collection application using API through Marvel Comics. It is a searchable api with with ability to search comic collection by:
Title
Author 
Series

Application gives the ability to:
create multiple collections
save comics too collection
view other user collections 
set collections to private or public 
view top comics of the week
